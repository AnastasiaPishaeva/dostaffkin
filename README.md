# Dostaffkin 
Dostaffkin — это веб-приложение для оформления заявок на доставку посылок и отслеживания их статуса по номеру заявки.
Пользователь может рассчитать параметры доставки, указав маршрут, характеристики посылки и желаемую скорость, а также оформить
заявку с контактными данными. После оформления доступно отслеживание статуса доставки (сборка, в пути и тд)
#### Демо: https://anastasiapishaeva.github.io/dostaffkin
---
## Основной функционал
#### Оформление доставки
- ввод адреса отправления и назначения
- выбор параметров посылки
- выбор скорости доставки
- ввод контактных данных
- добавление комментариев к заказу
#### Расчет стоимости и сроков
- расчет расстояния между точками
- определение примерного времени доставки
- расчет тарифа и итоговой стоимости
#### Интеграция карт
- построение маршрута доставки с использованием Яндекс Карт
- визуальное отображение пути между адресами
#### Отслеживание заказа
- получение информации о статусе доставки по номеру заявки

## Запуск проекта
```bash
# Установка зависимостей
npm install

# Запуск
npm start
```
Сайт будет доступен по адресу: `http://localhost:4200`

---
Настройка Яндекс Карт

Для работы карты используется API Яндекс Карт.
В файле index.html необходимо указать свои API-ключи:
```bash
<script
  src="https://api-maps.yandex.ru/2.1/?lang=ru_RU&apikey=YOUR_API_KEY&suggest_apikey=YOUR_SUGGEST_API_KEY"
  type="text/javascript">
</script>
```
Где взять ключи:
https://developer.tech.yandex.ru/services/
